Flashing Instructions

Prerequisites:

  • Unlocked bootloader
  • Custom recovery installed (SKKK TWRP for POCO F3, F4, MI 10T/Pro & OrangeFox for MI 11T Pro)
  • ADB and Fastboot tools on your PC
  • At least 50% battery
  • Backup of your important data

Installation Steps (Recovery):

  1. Format data if encrypted
  2. Flash Firmware:
    • 1.0.2.0 GLO/1.0.3.0 CN/1.0.1.0 IN for POCO F3
    • Latest regional HyperOS for POCO F4
    • Latest regional MIUI 14 for MI 10T/Pro
    • Latest global for MI 11T Pro
  3. Flash ROM .zip
  4. Wipe data, metadata, dalvik, cache
  5. Reboot System (Slot a)

Installation Steps (Fastboot):

  1. Extract ROM .zip
  2. Go to fastboot-installer folder
  3. Run the flash script depending on what OS you're on.

Important Notes:

  • All the ROMs here are decrypted but spoofed as encrypted, so no need to use DFE and Apps that needs encryption will work just fine.
  • Stock Kernel doesn't support any KSU Manager but supports KSU domain (for system scripts), Flash latest Oxygen+ KSU Variant to get KSU Manger Support
  • First boot may take 5-10 minutes.
