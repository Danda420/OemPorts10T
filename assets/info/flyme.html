FlymeOS

  • Only kernel with the same base as stock will boot.
  • This is a CN based rom so apps will be killed after its cleared from recents.. you need to turn off battery optimization for apps that you want to run in the background.. use activity launcher since this is a hidden settings.
  • Can't login google acc? Turn on google services from Settings > Accessibilty
  • Flyme dark mode forces dark mode for all apps.. it will definetly make apps looks gay.. disable it manually from Settings > Display >Night mode settings > Dark apps
